Map

Custos is using top of the line APIs to have the most accurate maps for every user. The Custos' development team is intertwining Google Maps with our unique built-in software that will magnify each user experience.

  • Continue

Events

Our users have the abilities to create an unlimited amount of events. Custos has created situational events that encompasses maximum protection.

  • Continue

Real Time Security

Custos has made it our responsibility in providing each user the ability to ensure love ones the location of their friends and families. We use real time location to assist every person to prevent any unfortunate scenarios.

  • Continue

Contacts

Custos has implemented a buddy system to provide a stronger network of security. Each user will have the ability to add or remove any other members in their friend's list. We anticipate this will bring users together to build a strong community.

  • Continue

Danger Zone

Our techincal team has invented a reporting system that will identify hazardous areas. Each danger zone, will have a severity level to indicate the seriousness of a threat that it poses.

  • Continue

Emergencies

Every individual will have a saftey net when it comes to emergencies. We created a saftey pin that will be prompted when they are not in a safe situation. Our application will also prompt each user to enter an emergency contact just in the off chance that there is a potential danger.
